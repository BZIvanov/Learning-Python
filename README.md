# Learning Python

This repository provides information and examples related to the Python programming language.

## Installation

You can download and install Python from the [official website](https://www.python.org/).

To check the currently installed version of Python in the terminal type `python --version`

## Official documentation

Refer to the official [docs](https://docs.python.org/3/) for detailed information.

## Content of this repository

It is recommended to follow this learning path in order:

1. **IntroToPython**
2. **VariablesAndBasicTypes**
3. **CompoundDataTypes**
4. **ControlFlow**
5. **Functions**
6. **ModulesAndPackages**
7. **FileHandling**
8. **ErrorAndExceptionHandling**
9. **OOP**
10. **IteratorsAndGenerators**
11. **Decorators**
12. **Tasks**
